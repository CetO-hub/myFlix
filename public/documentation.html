Documentation

The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Business logic Endpoint URL Method Request parameters Request format Response format
Return a list of ALL movies to the user http//:localhost:8080/movies GET none none A JSON object showing all movies
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user http//:localhost:8080/movies/:title GET :title none A JSON object holding data about a single movie
Return data about a genre (description) by name/title (e.g., “Thriller”) http//:localhost:8080/movies/:title/genre GET :title none A JSON object holding data about the genre of a single movie
Return data about a director (bio, birth year, death year) by name http//:localhost:8080/directors/:name GET :name none A JSON object holding data about all directors
Allow new users to register http//:localhost:8080/users POST none A JSON object with the respective data to be added (e.g. {firstName: "Tom", lastName: "Hardy", username: "Harom", email: "t.hardy@gmail.com", password: "ahsg1"}) A JSON object with the respective data (e.g. {id: 5, firstName: "Tom", lastName: "Hardy", username: "Harom", email: "t.hardy@gmail.com", password: "ahsg1"})
Allow users to update their user info (username) http//:localhost:8080/users/:id/:username PUT :id, :username none A text message that the username under the respective id was changed.
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added—more on this later) POST http//:localhost:8080/users/:title :title A JSON object holding the data for the movie (e.g. {title: "Matrix"}) A text message that the movie has been added
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later) DELETE http//:localhost:8080/users/:title :title A JSON object holding the data for the movie (e.g. {title: "Matrix"}) A text message that the movie has been deleted
Allow existing users to deregister (showing only a text that a user email has been removed—more on this later) DELETE http//:localhost:8080/users/:id :id A JSON object holding the data for the user (e.g. {id: "2"}) A text message that the user has been deleted
